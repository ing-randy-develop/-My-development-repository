1-Crear una Base de datos que se llame Shop_database.
2- Escribir el comando "upadate-database" en la consola del gestor de paquetes del proyecto.
3- En Visual Studio mandar a correr el proyecto(F5)
4- Para la autenticación los nombres de usuarios y contraseñas que se crean por defecto en la base de datos son: user: cliente pass: PassCliente123, 
user: admin pass: PassAdmin123, user: vendedor pass: PassVendedor123.
